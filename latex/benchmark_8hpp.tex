\doxysubsection{Detailed Description}
Wall-\/clock + cycle-\/accurate benchmarking for performance profiling. 

Wraps a simulation function and logs\+:
\begin{DoxyItemize}
\item Wall time in nanoseconds + seconds
\item CPU cycles (via rdtsc or cntvct\+\_\+el0)
\item π estimate from number of hits
