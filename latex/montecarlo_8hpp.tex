\doxysubsection{Detailed Description}
SIMD Monte Carlo Engine — Benchmarking π estimation via multiple memory and compute models. 

Demonstrates Monte Carlo Pi estimation using various high-\/performance methods\+:
\begin{DoxyItemize}
\item Sequential execution
\item Multithreading
\item SIMD vectorization (AVX2 / NEON)
\item Heap vs Pool memory allocation
\end{DoxyItemize}

Originally extended from a Monte Carlo Pi simulation assignment for CSULB CECS 325 (Systems Programming taught by Neal Terrell during Spring 2025), adapted by Leon.\hypertarget{montecarlo_8hpp_autotoc_md5}{}\doxysubsubsection{\texorpdfstring{Features}{Features}}\label{montecarlo_8hpp_autotoc_md5}

\begin{DoxyItemize}
\item Supports both x86 (AVX2) and ARM (NEON) SIMD.
\item Pluggable {\ttfamily \doxylink{structPoolAllocator}{Pool\+Allocator}} for thread-\/local fast memory.
\item Ideal for comparing memory models in simulation-\/heavy workloads.
\end{DoxyItemize}\hypertarget{montecarlo_8hpp_autotoc_md6}{}\doxysubsubsection{\texorpdfstring{Implementations}{Implementations}}\label{montecarlo_8hpp_autotoc_md6}

\begin{DoxyItemize}
\item {\ttfamily monte\+Carlo\+PI\+\_\+\+SEQUENTIAl} — Scalar baseline.
\item {\ttfamily monte\+Carlo\+PI\+\_\+\+HEAP} — Heap-\/based allocation.
\item {\ttfamily monte\+Carlo\+PI\+\_\+\+POOL} — Bump-\/allocated thread-\/local memory.
\item {\ttfamily monte\+Carlo\+PI\+\_\+\+SIMD} — Vectorized simulation using AVX2/\+NEON.
\end{DoxyItemize}\hypertarget{montecarlo_8hpp_autotoc_md7}{}\doxysubsubsection{\texorpdfstring{Requirements}{Requirements}}\label{montecarlo_8hpp_autotoc_md7}

\begin{DoxyItemize}
\item {\ttfamily \doxylink{pool_8hpp}{pool.\+hpp}} — bump allocator used in pool-\/based variants.
\item SIMD supported hardware.
\item C++17 or higher for {\ttfamily std\+::aligned\+\_\+alloc} used in \doxylink{pool_8hpp}{pool.\+hpp}.
\end{DoxyItemize}

Compilation fails gracefully if SIMD is not supported.
