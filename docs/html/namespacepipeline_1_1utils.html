Safely performs division, handling 'NA' values and invalid input.

Returns a rounded division result unless input is invalid or contains the string "NA", in which case "NA" is returned instead.

Cast a Polars DataFrame to match a declared schema, handling 'NA' strings as nulls.

This function enforces schema alignment between a raw input DataFrame (typically from CSV) and a declared schema. If allow_na is True in the schema, string values like "NA" will be replaced with nulls prior to casting.

Parameters
dfThe input Polars DataFrame to cast.
schemaDictionary in the format { column_name: (dtype, allow_na) }.
Returns
A new Polars DataFrame with all fields casted according to the schema.
Exceptions
ValueErrorIf any schema field is missing in the DataFrame.
